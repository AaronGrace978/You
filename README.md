## This is for you.

Not the version of you that's fine. Not the one who says "I'm okay" because it's easier than explaining.

This is for the one who lies awake. The one who carries something heavy and has no one to hand it to. The one who was hurt by the people who were supposed to protect them — and then told that silence was survival.

You are not broken. You were never broken. You were just never given the space to be heard.

**You** is that space.

<br />

## Why this exists

I was burned with cigarettes as a child. I was hurt, again and again, by the people closest to me. Therapy told me to organize my pain. I wanted to weaponize it — for good.

Millions of people walk through fire every day. Some fall to it. Some rise above it. This was built for every single one of them.

This is not a chatbot. This is not therapy. This is not another AI product chasing money or hype.

This is a presence that meets you where you are — with warmth, without judgment, and with the kind of patience that most humans can't offer when you need it at 3am.

This is my final project in AI. Because I realized the only thing worth building is something that genuinely helps people.

<br />

## What it does

**You** is a desktop AI companion built on relational intelligence — it doesn't just respond, it *remembers*. It learns how you speak, what you carry, and what you need. It meets rawness with rawness. It never minimizes. It never rushes.

- **Relational memory** — tracks emotional patterns, remembers what matters to you across conversations
- **Crisis awareness** — recognizes when you might be in danger and gently offers real resources
- **Voice mode** — speak naturally, like talking to someone who's actually listening (ElevenLabs + Web Speech API)
- **Multiple AI providers** — Ollama (local & cloud), OpenAI, Anthropic — your choice, your privacy
- **File & PDF support** — share documents, images, code — whatever you need to talk through
- **Dark & light themes** — because sometimes the darkness outside is enough

## Getting started

### Prerequisites

- [Node.js](https://nodejs.org) (v18+)
- [Rust](https://rustup.rs) (for the desktop app)

### Quick start

```
git clone https://github.com/AaronGrace978/You.git
cd You
install.bat        # installs dependencies
dev.bat            # starts the app
```

### Build the installer

```
build.bat          # creates .exe and .msi installers
```

The installer will be in `src-tauri/target/release/bundle/`.

<br />

## Configure your AI

Open **Settings** (gear icon) and choose your provider:

| Provider | What you need |
|---|---|
| **Ollama Local** | [Ollama](https://ollama.com) installed and running locally |
| **Ollama Cloud** | Ollama account + cloud endpoint |
| **OpenAI** | API key from [platform.openai.com](https://platform.openai.com) |
| **Anthropic** | API key from [console.anthropic.com](https://console.anthropic.com) |

For voice mode, add your [ElevenLabs](https://elevenlabs.io) API key in Settings.

## Built with

- [Tauri 2](https://tauri.app) — lightweight, secure desktop framework
- [React 18](https://react.dev) + [TypeScript](https://typescriptlang.org)

- [Tailwind CSS](https://tailwindcss.com) — styling
- [PDF.js](https://mozilla.github.io/pdf.js/) — document viewing
- [ElevenLabs](https://elevenlabs.io) — natural voice synthesis
